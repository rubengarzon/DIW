TUTORIAL CSS3 - Transiciones

Propiedad que permite definir el cambio de estilo entre dos estados de los elementos de un documento HTML

transition-delay

Indica cuando comienza el efecto de la transicion

Sintaxis

transition-delay: time;

Admite de un valor:

  • time: indica el tiempo que se tiene que esperar antes de que se inice la transicion.

Ejemplos:


.transition-delay{
    width: 100px;
    height: 100px;
    background-color: #0000FF;
    transition-delay: 2s;
    }
.transition-delay:hover {
    background-color: #FFCCCC;
    width: 200px;
    height: 200px;
    }

transition-duration

especifica cuántos segundos o milisegundos tarda en completarse un efecto de transición..

Sintaxis

transition-duration: time;

Admite de un valor:

  • time: indica el tiempo en segundos o milisegundos que llevara hasta completar la transicion.

Ejemplos:

 
.transition-duration{
    width: 100px;
    height: 100px;
    background-color: #0000FF;
    transition-duration: 3s;
    }
.transition-duration:hover {
    background-color: #FFCCCC;
    width: 200px;
    height: 200px;
    }

Transition-property

Se usa para definir los nombres de las propiedades CSS en las que el efecto de la transición debe aplicarse.

Sintaxis

transition-property: width;

Contiene los siguientes valores:

  • None: ninguna propiedad tendrá transición.
  • All: Todas las propiedades que puedan tener una transición animada la tendrán.
  • IDENT: Una cadena que identifique la propiedad en la que el efecto de la transición debe ser aplicado cuando su valor cambie.

Ejemplos:

div {
    width: 100px;
    height: 100px;
    background: red;
    transition-property: width;
    transition-duration: 2s;
}
                  
div:hover {
    width: 300px;
}

Transition-timing-function

Especifica la curva de velocidad del efecto de transición.

Sintaxis

transition-timing-function: linear;

Contiene los siguientes valores:

  • Ease: Igual al cubic-bezier(0.25, 0.1, 0.25, 1.0)valor predeterminado, aumenta la velocidad hacia la mitad de la transición, disminuyendo la velocidad al final.
  • linear. Igual que cubic-bezier(0.0, 0.0, 1.0, 1.0), transiciones a una velocidad uniforme..
  • ease-in: Igual a cubic-bezier(0.42, 0, 1.0, 1.0), comienza lentamente, con la velocidad de transición aumentando hasta completarse.
  • ease-out: Igual que cubic-bezier(0, 0, 0.58, 1.0), comienza la transición rápidamente, la transición continúa ralentizándose.
  • ease-in-out: Igual a cubic-bezier(0.42, 0, 0.58, 1.0), comienza a hacer la transición lentamente, acelera y luego vuelve a disminuir.
  • cubic-bezier(p1, p2, p3, p4): Un autor definió la curva cúbica-Bézier, donde los valores p1 y p3 deben estar en el rango de 0 a 1.
  • steps( n, ): Muestra la transición a lo largo de n paradas a lo largo de la transición, mostrando cada parada durante períodos de tiempo iguales. Por ejemplo, si n es 5, hay 5 pasos.
  • jump-start: Denota una función continua a la izquierda, de modo que el primer salto ocurre cuando comienza la transición.
  • jump-end: Denota una función continua a la derecha, de modo que el último salto ocurre cuando termina la animación;
  • jump-none: se aplica el primero a arriba, el segundo a la derecha, el tercero abajo y el cuarto a la izquierda.
  • jump-both: Incluye pausas en las marcas 0% y 100%, agregando efectivamente un paso durante el tiempo de transición.
  • start: Igual que jump-start.
  • end: Igual que jump-end.
  • step-start: Igual a steps(1, jump-start)
  • step-end: Igual a steps(1, jump-end)

Ejemplos:


.ease {
    transition-timing-function: ease;
}
.easein {
    transition-timing-function: ease-in;
}
.easeout {
    transition-timing-function: ease-out;
}
.easeinout {
    transition-timing-function: ease-in-out;
}
.linear {
    transition-timing-function: linear;
}

transition

Es una propiedad con shorthand que se puede usar para agrupar proiedades, las propiedades que acepta son: transition-property, transition-duration, transition-timing-function y transition-delay

Sintaxis

transition: property duration timing-function delay;

Admite de uno a cuatro valores, que son los siguientes:

  • transition-property: espedifica el nombre de la propiedad CSS sobre la que se hara la transicion.
  • transition-duration: especifica el tiempo el segundos o milisegundos que durara la transicion
  • transition-timing-function: da la velocidad de la curva de la transicion
  • transition-delay: define cuando empezara la transicion

Ejemplos:


.transition{
    width: 100px;
    height: 100px;
    background-color: #0000FF;
    transition: width 2s, height 2s, background-color 2s, transform 2s;
    }
.transition:hover {
    background-color: #FFCCCC;
    width: 200px;
    height: 200px;
    }
